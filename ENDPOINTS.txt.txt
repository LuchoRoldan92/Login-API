                                    ENDPOINTS DE LA API


  ---- Descripción General----

La API fue desarrollada utilizando Node.js y Express para gestionar el registro e inicio de sesión de usuarios.




                         ---> Endpoint 1: Registro de Usuario <---

--Método HTTP

    POST

--URL

    http://localhost:3000/registro

--Descripción

    Permite registrar un nuevo usuario en el sistema.

--Datos de Entrada (JSON)

    {
      "nombre": "Luis",
      "password": "123456"
    }

    --- Respuesta Exitosa ---

    {
      "mensaje": "Usuario registrado correctamente"
    }

                           ---> Endpoint 2: Inicio de Sesión <---

--Método HTTP

     POST

--URL

     http://localhost:3000/login

--Descripción

     Permite autenticar un usuario previamente registrado.

--Datos de Entrada (JSON)

    {
     "usuario": "Luis",
     "password": "123456"
    }

--Respuesta Exitosa

    {
     "mensaje": "Inicio de sesión exitoso"
    }
